# Autenticación con Tokens - NestJS + Angular

Este proyecto es una aplicación full-stack que implementa un sistema de autenticación basado en **tokens JWT** usando **NestJS** en el backend y **Angular** en el frontend.

## 🧰 Tecnologías Utilizadas

- **Backend:** [NestJS](https://nestjs.com/) (Node.js + TypeScript)
- **Frontend:** [Angular](https://angular.io/)
- **Base de datos:** PostgreSQL (Supabase)
- **Autenticación:** JWT (JSON Web Tokens)
